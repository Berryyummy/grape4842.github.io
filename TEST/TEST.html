스크린리더 테스트를 위한 요소,기타 잡다한 것을 일시적으로 마크업 한 페이지입니다.

텍스트등의 주된 정보를 이미지로 제공하는 경우의 문제점.
문제
이미지 텍스트
한 번에 많은 내용이 제공된 경우, 대체텍스트(alt)는 스크린리더로 끊어 읽을 수 없다.
프로그램이 읽고 이해할 수 없기 때문에 자동번역,TTS출력등이 불가능하다.
대체텍스트가 올바르게 제공되지 않은 경우, 내용을 파악할 수 없다.
돋보기와 같은 확대 기능을 사용할 경우, 종횡 양측에 스크롤이 생성되어 정보를 한번에 열람하기 어렵다.
확대기능 사용자와 저시력 사용자의 불편 사항
문제
확대/크기조정 글자,화면을 크게 설정하고 사용하는 경우, 앱이 대응(사용자의 설정과 무관하게 고정된 크기로 설정했거나 유동적인 크기 변경되도록 제작)하지 않은 경우, 화면의 레이아웃이 꺠져 앱을 정상적으로 이용하기 어렵다.
PC에서 돋보기로 화면을 확대하여 사용하는 경우 마우스 호버 이벤트로 열리고, 마우스가 영역 내에 있는 동안만 열려있는 메뉴의 경우, 주변 탐색을 위해 마우스가 해당 영역을 벗어나게 되면, 이벤트가 해제되어 메뉴가 닫혀 자세한 내용을 확인할 수 없음.
위와 동일한 이유로 마우스 호버로만 확인할 수 있는 툴팁 또한 내용을 확인하기 어려움.
그 외 아이콘만으로 기능을 설명하는 경우, 아이콘의 윤곽이 뚜렷히 보이지 않아 기능을 이해하지 못할 수 있음.
입력서식이 label과 연결되지 않은 경우, 입력서식을 직접 터치해야 하기 때문에 입력서식의 위치를 정확히 찾아 누르는 것이 어려움.
디스플레이 옵션의 각 명칭과 특징
열:각 모드
행:설명항목 		고대비모드 다크모드 색반전 색각 배려
정의 사용자가 각 요소(배경과 텍스트, 아이콘과 배경, 요소 테두리와 배경)간을 명도대비가 높은 색상(검정색:흰색, 검정색:노란색 등)으로 색 체계를 재구성하여 표시하는 설정 사용자가 어두운 환경에서 기기를 사용할 경우, 빛으로 인해 눈이 부시지 않도록 어두운 색 체계로 재구성하여 표시하는 설정
  • 사용자가 보는 화면을 반대되는 색상으로 표시하게 되어, 밝은 색은 어둡게, 어두운 색은 밝게 뒤집어 표시하는 설정
  • (iOS에 제공되는 스마트 색반전의 경우 멀티미디어(이미지,동영상)등을 제외하여 반전됨.)
 사용자가 색약,색맹이라 하더라도 색상에 상관없이 각 컨텐츠의 경계를 구별할 수 있도록 색 체계를 재구성하여 표시하는 설정
특징 및 장단점
  • 각 요소간의 경계가 색상과 상관없이 명도대비로 구별되어 색을 인지하지 못하여도 경계를 뚜렷히 구분할 수 있게 된다.
  • 각 요소의 경계가 뚜렷하므로 요소의 위치나 영역을 정확히 인지하고 선택,조작할 수 있게 된다.
  • 개발자 혹은 사용자가 직접 색상을 재구성하는 것이므로 이미지,동영상과 같은 멀티미더어 요소가 색상 변화의 영향을 받지 않는다.
  • 고대비 모드의 작동 원리상, 색체계를 강제적으로 재구성하는 것이기 때문에, 개발자가 지정하지 않은 경우, 혹은 충돌이 일어난 경우, 정상적으로 컨텐츠를 인식할 수 없게 잘못 표시되거나, 고대비모드가 적용되지 않을 수 있다.
  • 저조도 환경에서 사용하는 사용자,밝은 화면을 보기 어려워하는 사용자는 전체적으로 어두운 색으로 변경된 화면이므로, 밝기를 줄인 것과 달리 화면을 쾌적하게 사용할 수 있다.
  • 사용자에 따라서는 어두운 배경에 밝은 글씨를 더 인지하기 쉬운 경우가 있으므로, 그러한 사용자의 경우, 사용성이 증진한다.
  • 어두운 배경에 밝은 글씨는 높은 명암비를 갖고 있기 때문에 특정 정보를 강조할 때는 유리하다.
  • 읽는 시간이 길어지거나 정확히 읽어야 할 때는 밝은 배경에 어두운 글씨가 더 좋다. 이는 단순히 우리가 밝은 배경, 어두운 글씨에 익숙하기 때문이 아닌, 어두운 배경에서는 동공이 열려 텍스트에 집중하는 것이 어려워진다는 연구결과이다.
  • 개발자 혹은 사용자가 직접 색상을 재구성하는 것이므로 이미지,동영상과 같은 멀티미더어 요소는, 색상 변화의 영향을 받지 않는다.
  • OLED를 사용하는 디스플레이의 경우
    • 밝기가 낮거나 어두운 색상으로 이루어진 화면은 자체적으로 발광하는 OLED의 특성상 전력소모가 줄어들게 되어 배터리를 절약할 수 있다.
      • \
    • 완전한 블랙으로 배경을 제공하는 경우, 픽셀을 끄기 때문에 플리커 현상을 감소시켜 눈의 피로를 상대적으로 줄일 수 있다.
    • 어두운 색상을 사용하므로 결과적으로 동시간 대비 픽셀의 소비가 줄어 번인을 늦출 수 있다.
  • 필터와 같이 화면에 색상을 그대로 뒤집는 것이기 때문에 호환성이 높다.
  • 필터를 화면 전체에 덮는 개념이기 떄문에 모든 요소가 반전되므로, 이미지 및 동영상과 같은 매체를 정상적으로 이용할 수 없다.
  • 색을 단순히 반전시키는 것이므로 명도대비의 개선은 기대할 수 없다.
  • 스마트 반전의 경우, 일반적인 색상 반전과 달리, 영상이나 이미지 등을 반전하지 않기 때문에 고대비모드,다크모드가 제공되지 않는 경우, 대체제로 고려할 수 있다.
