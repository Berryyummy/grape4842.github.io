정의 사용자가 각 요소(배경과 텍스트, 아이콘과 배경, 요소 테두리와 배경)간을 명도대비가 높은 색상(검정색:흰색, 검정색:노란색 등)으로 색 체계를 재구성하여 표시하는 설정 사용자가 어두운 환경에서 기기를 사용할 경우, 빛으로 인해 눈이 부시지 않도록 어두운 색 체계로 재구성하여 표시하는 설정 사용자가 보는 화면을 멀티미디어(이미지,동영상)를 제외하고, 색을 반대되는 색상으로 표시하는 설정 사용자가 보는 화면을 반대되는 색상으로 표시하는 설정
특징
  • 각 요소간의 경계가 색상과 상관없이 명도대비로 구별되어 색을 인지하지 못하여도 경계를 뚜렷히 구분할 수 있음.
  • 각 요소의 경계가 뚜렷하므로 요소의 위치나 영역을 정확히 인지하고 선택,조작할 수 있음.
  • 밝은 화면을 보기 어려워하는 사용자는 전체적으로 어두운 색으로 변경된 화면이므로, 밝기를 줄인 것과 달리 화면을 쾌적하게 사용할 수 있음.
  • 사용자에 따라 다를 수 있으나 같은 명도대비 조건의 경우, 어두운 배경에 밝은 글씨를 더 인지하기 쉬운 경우가 있으므로, 그러한 사용자의 경우, 사용성이 증진함.
  • OLED를 사용하는 전자기기의 경우, 밝기가 낮거나 어두운 색상으로 이루어진 화면은 자체적으로 발광하는 OLED의 특성상 전력소모가 줄어들게 되어 배터리를 다소 절약할 수 있음.
